Fybrik

A cloud-native platform to control data usage

Get started Go to GitHub

Fybrik is a cloud native platform to unify data access, governance and orchestration, enabling business agility while securing enterprise data. By providing access and use of data only via the platform, Fybrik brings together access, performance and governance for data, greatly reducing risk of data loss.

Connect

Abstract the connection to data and allow seamless access agnostic to the locations of computation and data or the data API and protocol

Secure

Secure data usage and enforce governance by intermediating all input and output flows

Control

Control and optimize the workload behavior by orchestrating data functions and data mobility. Optimize performance using technologies such as caching, copies and more

Observe (future)

Provides instrumentation to observe the use of data by enabling the seamless platform-level collection of metrics, logs, audit and data provenance information
